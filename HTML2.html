HTML Styles
The HTML style attribute is used to add styles to an element, such as color, font, size, and more.
Example::
                               I am Indian
                               I am  Indian
                              I am Indian

HTML styles:

Background Color
The CSS background-color property defines the background color for an HTML element.
example:
< body style="background-color:rgb(12, 187, 35);">
< h1>This is a heading

< p style="background-color: tomato;">This is a paragraph.

< /body>
< /body>


Text Color
The CSS color property defines the text color for an HTML element:

This is a heading

This is a paragraph.



Fonts
The CSS font-family property defines the font to be used for an HTML element < h1 style="font-family:verdana;">This is a heading < p style="font-family:courier;">This is a paragraph.



Text Alignment
The CSS text-align property defines the horizontal text alignment for an HTML element < h1 style="text-align:center;">Centered Heading < p style="text-align:center;">Centered paragraph.



HTML Formatting Elements
Formatting elements were designed to display special types of text:

            < b>- Bold text This text is bold
            < strong>- Important text>This text is important!
            < i>- Italic text>This text is italic
            < em>- Emphasized text>This text is emphasized
            < mark> - Marked text> Do not forget to buy milk today. 
            < small> - Smaller text>This is some smaller text.
            < del>- Deleted text> My favorite color is blue red. 
            < ins>- Inserted text> My favorite color is blue red. 
            < sub>- Subscript text> This is subscripted text. 
            < sup>- Superscript text > This is superscripted text.
